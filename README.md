Sales Performance Dashboard 2022 created using Microsoft Excel.
This dashboard shows total sales, units sold, payment modes, top products, and top sellers.

Charts and graphs are used to visualize the sales trend for the last 28 days and previous 28 days.

Dashboard 2 – Sales Analytics Dashboard

This dashboard focuses on sales analytics and revenue insights. It displays total revenue, total sales, product-wise sales, and sales trends over time. Visual charts and graphs help in understanding business performance and identifying top-performing products.
